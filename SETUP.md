# Настройка Telegram Mini App — Дизайн-профилирование

## 1. Захостить файл на GitHub Pages

### Шаг 1: Создать репозиторий
1. Зайди на [github.com/new](https://github.com/new)
2. Название: `design-profiling` (или любое другое)
3. Видимость: **Public** (для GitHub Pages на бесплатном плане)
4. Нажми **Create repository**

### Шаг 2: Загрузить файл
1. В репозитории нажми **Add file → Upload files**
2. Загрузи `index.html`
3. Нажми **Commit changes**

### Шаг 3: Включить GitHub Pages
1. Перейди в **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main`, папка: `/ (root)`
4. Нажми **Save**
5. Через 1–2 минуты сайт будет доступен по адресу:
   ```
   https://<твой-username>.github.io/design-profiling/
   ```

## 2. Создать Telegram-бота

### Шаг 1: Зарегистрировать бота
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`
3. Введи название бота (напр. `Дизайн Профилирование`)
4. Введи username бота (напр. `design_profiling_bot`) — должен заканчиваться на `bot`
5. Сохрани токен (понадобится, если будешь расширять функциональность)

### Шаг 2: Создать Web App
1. В чате с @BotFather отправь `/newapp`
2. Выбери своего бота
3. Введи название: `Дизайн-профилирование`
4. Введи описание: `Интерактивный тест для определения дизайн-специальности`
5. Загрузи фото (640×360 px) — обложка приложения
6. Отправь GIF-анимацию или напиши `/empty` чтобы пропустить
7. Введи URL:
   ```
   https://<твой-username>.github.io/design-profiling/
   ```
8. Введи short name: `test` (или другое короткое имя)

### Шаг 3: Настроить кнопку в меню бота
1. Отправь @BotFather команду `/setmenubutton`
2. ВыЛери своего бота
3. Введи URL:
   ```
   https://<твой-username>.github.io/design-profiling/
   ```
4. Введи текст кнопки: `Пройти тест`

Теперь при открытии бота внизу будет кнопка **«Пройти тест»**, которая запускает Mini App.

## 3. Альтернативный запуск

### Прямая ссылка
Можно отправлять пользователям ссылку вида:
```
https://t.me/<bot_username>/test
```
где `test` — short name, указанный при создании Web App.

### Inline-кнопка
Если бот отправляет сообщения, можно добавить inline-кнопку:
```json
{
  "text": "Пройти тест",
  "web_app": { "url": "https://<username>.github.io/design-profiling/" }
}
```

## 4. Проверка

1. Открой бота в Telegram на телефоне
2. Нажми кнопку «Пройти тест»
3. Приложение должно открыться в полноэкранном режиме
4. Кнопка «Назад» (BackButton) — работает на всех экранах кроме первого
5. На экране результатов появится кнопка «Закрыть» (MainButton)
